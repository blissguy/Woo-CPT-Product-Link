Link custom post type entries to WooCommerce products and render product data from CPT templates.

== Description ==

Woo CPT Product Link lets selected public custom post types reference real WooCommerce products. It adds an editor product selector, frontend shortcodes, and Bricks dynamic data tags for product pricing, product details, product type, and whether a related product exists.

== Installation ==

1. Upload the `woo-cpt-product-link` folder to `/wp-content/plugins/`.
2. Activate Woo CPT Product Link in WordPress.
3. Open WooCommerce > Woo CPT Product Link and choose which public post types can link products.
